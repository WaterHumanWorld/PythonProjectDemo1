# 自动化 图形界面打包程序 应用
安装Auto-Py-To-Exe：

pip install auto-py-to-exe
CopyInsert
启动Auto-Py-To-Exe：

auto-py-to-exe
CopyInsert
配置和打包： 在打开的Web界面中，你可以选择你的Python脚本文件，配置打包选项（如单个文件、图标、隐藏控制台等），然后点击“CONVERT .PY TO .EXE”按钮来生成可执行文件。

Auto-Py-To-Exe GitHub页面
cx_Freeze官方文档
